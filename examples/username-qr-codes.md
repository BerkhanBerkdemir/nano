# Generate QR Code PNG API

### HTML Examples

**Basic Example**


```html
<a href="https://nano.to/Moon">
  <img style="max-width: 100px" src="https://nano.to/qrcode/Moon"/>
</a>
```

<img style="max-width: 100px" src="https://nano.to/qrcode/Moon"/>

```html

```
